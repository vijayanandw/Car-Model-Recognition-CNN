# Car Model Recognition and Classification using CNN

A deep learning-based web application that identifies car brands and models from images using Convolutional Neural Networks (CNN). Built with Flask, TensorFlow/Keras, and OpenCV for accurate vehicle classification through a user-friendly web interface.

## Features
- Car brand and model recognition
- CNN-based image classification
- Flask web application
- Real-time image prediction
- Responsive user interface

## Technologies Used
- Python
- TensorFlow / Keras
- Flask
- OpenCV
- HTML, CSS, JavaScript

```bash
pip install -r requirements.txt
python app.py
